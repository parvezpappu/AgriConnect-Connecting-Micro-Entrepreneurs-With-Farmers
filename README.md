# AgriConnect — Connecting Micro Entrepreneurs with Farmers

AgriConnect is a web application designed to improve agricultural supply chains in developing regions by connecting **Farmers **And** ShopOwner** in one platform. The system addresses common issues like limited market access, price exploitation by middlemen, and lack of resources—while enabling micro-entrepreneurs/hub employees to coordinate delivery and order fulfillment.

## Project Overview

### Goals & Objectives
- Improve farmers’ access to fair markets and buyers  
- Reduce dependency on middlemen and improve pricing transparency  
- Enable micro-entrepreneurs (Employees) to support logistics and distribution  
- Provide an organized, role-based system for product listing, ordering, payment, and delivery tracking  




### Access Layers (Roles)
- **Admin**
- **Farmer**
- **Buyer**
- **Employee**

---

## Core Features

### 1) Authentication System
- User Sign-up (Farmer/Buyer/Employee) with OTP verification  
- Login (email/phone + password) with role-based dashboards   
- Sign-out with session/token invalidation  

---

### 2) Dashboards
- **Admin Dashboard
- **Farmer Dashboard:** manage products,orders  
- **Buyer Dashboard:** recent orders
- **Employee Dashboard:** assigned orders

---

### 3) User Profile Management
- View Profile 
- Edit Profile  
- Update Contact )
- Change Password (old password required + secure update)  

---

### 4) Product Management
- Add Product (name, price, stock, images) — Admin/Employee  
- Update Product — Admin/Employee  
- Delete Product — Admin/Employee (soft/hard delete)   
- Product Analytics (sales + stock stats) — Admin/Farmer  

---

### 5) Product Browsing (Buyer)
- Product List with pagination  
- Search by keyword  
- Filters (category, price, stock)  
- Category-wise pages  
- Product Details (images + full info + add to cart)  
- New Arrivals (badge based on creation date)

---

### 6) Cart & Checkout (Buyer)
- Add/Remove items in cart  
- Change quantity (+/-) with stock validation  
- Auto price calculation (subtotal, VAT, total, shipping if any)  
- Save cart (persistent cart across sessions)  
- Mini cart quick-view dropdown

---

### 7) Order Management
- Place Orders (order ID generation, stock reduction)  
- Order Tracking (real-time status timeline)  
- Cancel/Modify (allowed only before shipping)  
- Order History (search/sort + details)  
- Hub Assignment (admin/employee assigns orders to hubs)

---

### 8) Payment System
- bKash Payment (simulated/mobile wallet)  
- Nagad Payment (simulated)  
- Bank Transfer (manual verification + proof upload)  
- Payment Confirmation (receipt + admin verification)  
- Payment Failure Handling (retry without duplicate charges)
